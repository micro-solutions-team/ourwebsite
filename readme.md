# MicroSolutions Website

MicroSolutions is a small, flexible technical team specializing in rapid delivery of **micro-solutions**, integrations, and AI-driven tooling for enterprise environments.  

This repository hosts the **corporate website** showcasing our services, sample projects, and technical capabilities.

## Features

- Corporate landing page with team overview and services
- Integration with GitHub repositories to showcase micro-solutions
- Mobile-friendly, responsive design
- SEO-ready with Open Graph and Twitter metadata
- Full favicon and PWA support

## Tech Stack

- **Frontend:** React + JSX
- **Styling:** CSS / Tailwind (optional)
- **Deployment-ready:** Vite build, static hosting compatible
